# Basement — 프로젝트 공통 지침서

이 폴더는 모든 프로젝트에서 공통으로 적용할 행동 규칙과 스킬 문서를 보관합니다.

## 사용 방법

새 프로젝트의 `CLAUDE.md`에 아래와 같이 추가하면 해당 규칙이 자동으로 적용됩니다:

```markdown
@/Users/park/Documents/project/Basement/rules/skill-learning.md
```

## 스킬 문서 충돌 정책

각각의 프로젝트 폴더 내에 Basement와 동일한 이름의 스킬 문서가 존재할 경우:

1. **즉시 덮어쓰지 않는다.** 두 문서를 먼저 비교한다.
2. **더 충실한 문서를 우선 적용한다.** 판단 기준은 다음과 같다:
   - 규칙이 더 구체적이고 예시가 풍부한 쪽
   - 금지 사항 또는 예외 조건이 더 명확하게 정의된 쪽
   - 실제 프로젝트 맥락을 반영한 내용이 추가된 쪽
3. **프로젝트 문서가 더 충실하다면**, 해당 내용을 Basement에 반영할 것을 사용자에게 제안한다.
4. **Basement 문서가 더 충실하다면**, 프로젝트 문서를 Basement 기준으로 보완할 것을 제안한다.
5. **두 문서가 서로 다른 내용을 다루고 있다면**, 충돌이 아닌 보완 관계로 보고 둘 다 적용한다.
