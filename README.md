# 마이크로팩토리 — 맥주 공정 시뮬레이터

## 작업의 배경
- “손 안의 마이크로팩토리” 콘셉트로, 맥주 제조 전 과정을 시각화하고 단계별 의사결정을 직관적으로 돕는 웹 데모입니다.
- 초보자는 공정의 의미를 이해하고, 숙련자는 파라미터를 조정하며 레퍼런스(시판 맥주)와 비교해 차이를 빠르게 파악하는 데 목적이 있습니다.
- 랜딩(Home)에서는 맥주 색/거품/기포 등 맥주 메타포를 활용해 스크롤 기반 스토리텔링으로 서비스 가치를 소개합니다.

## 메뉴별 주요 목적 및 기능
### Home
- 스크롤 연출(Scrollytelling)로 “제조 탐험 → 공정 → 시뮬레이션 → 상용화” 흐름을 간단히 안내
- 액체(맥주) 배경 애니메이션과 기포/거품 효과로 몰입감 강화
- “시뮬레이터 시작” 진입 CTA 제공

### Simulator
- 공정 파이프라인: 상단 가로 타임라인에서 현재 위치(활성/완료 단계)를 한눈에 인지
- 레퍼런스 비교: 좌측 리스트에서 국가·브랜드·맥주명을 선택(일부는 실제 로고 표시), 우측 지표에 기준치(고스트 마커) 표시
- 실시간 지표: ABV(도수), IBU(쓴맛), Body(바디감)를 게이지로 시각화(기준치 vs 내 레시피)
- 시각화: 우측 “맥주잔”에 단계별 탁도(blur)·색(SRM 유사)·거품 변화 반영
- 포장: 패키징 단계에서 병/캔/케그 시각을 실시간 상태 영역에서 미리보기
- 조작성: 모든 파라미터는 스냅핑 가능한 슬라이더(눈금/단위 표시)로 직관 조작
- 반응형: 데스크톱 3단(좌 리스트/중 상세/우 결과) → 태블릿/모바일 2·1단 자동 전환

## 시뮬레이터 단계별 내용(표준 9단계)
1. 원료 분쇄: 롤러 간격(0.5–2.0mm, 0.1mm 간격). 좁을수록 추출↑/여과 난이도↑  
2. 당화(맥아 끓이기): 온도·시간. 온도↓ → 발효도↑ → ABV↑·바디감↓ / 온도↑ → ABV↓·바디감↑  
3. 여과(찌꺼기 분리): 맑은 맥즙만 분리. 시각화에서 탁도↓  
4. 홉 끓이기: 끓임 시간. 길수록 IBU↑(쓴맛 강화)  
5. 맥즙 냉각: 효모 투입 전 목표 온도까지 빠르게 냉각  
6. 발효: 효모 온도·일수. 당이 알코올·탄산으로 전환(ABV·바디 감각에 영향)  
7. 숙성: 맛 정제·불순물 침전  
8. 탄산 주입: CO₂ 레벨(vol) 설정으로 청량감 튜닝  
9. 패키징: 병/캔/케그 선택 시 결과 영역에서 형태가 즉시 반영

## 레퍼런스 비교(참고 맥주)
- 좌측 리스트에서 국가/국기·브랜드·맥주명을 선택해 기준 레시피를 설정
- 각 단계 슬라이더에는 기준값(고스트 마커)이 함께 표시되어 내 레시피와 차이를 직관 비교
- 일부 브랜드는 실제 엠블럼을 표시(투명 배경 우선). 로고 미확보 시 모노그램 폴백

## 데이터 구조 개요
- `data/seed.json`  
  - styles: 스타일별 범위(예: APA의 IBU/ABV 범위)  
  - steps: 표준 9단계와 파라미터 스키마(범위/단위/기본값)  
  - recipes: 기본 레시피(배치 크기, 곡물/홉/효모, 단계 파라미터)

## 향후 계획(로드맵)
- 레퍼런스 로고: 각 브랜드의 공식 에셋/미디어 키트 출처로 투명 배경 로고 연결 확대(미확보 시 폴백 유지)
- 비교 시각화 강화: 방사형(레이더) 차트로 기준·내 레시피의 풍미 프로필 직관 비교
- 상용화 보드: 원가/유통/패키징 캔버스 재구성 및 공유/내보내기 지원
